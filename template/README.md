# template/

Reserved, intentionally empty in v1.0.0.

This project's chapters are hand-authored, pre-built standalone HTML
files (see the hard rule "DO NOT regenerate any chapter" in the
project's original brief) — they are never generated from a shared
template. This directory exists to match the originally-specified
project structure and is reserved for a possible future template-driven
authoring workflow (see TODO.md, v2 candidate). Nothing in the v1.0.0
pipeline reads from or writes to this directory.
